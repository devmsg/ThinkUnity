# ThinkUnity

更多偏向UI的构建工程



---


Assects目录结构：

```
|-- Common                     /管理后台URL重定向目录，你可以将文件夹名改为任何你喜欢的
    |--Config                  /后台模板文件目录
|--Resources                 /应用目录
|--Scenes                       /各类数据存放目录，包括缓存数据
|--Scripts                  /核心包，无特殊情况请勿改动
    |--Controller 
        |--Home
    |--Model
    |--View 
        |--HomeView
            |--Config
```
