Assects目录结构：

```
|-- Common                                      /管理项目公共模块的变量目录
    |--Config                                   /用于存放项目公共模块中所有文本信息变量
|-- Plugins                                        /插件目录
|-- Resources                                   /资源目录
|-- Scenes                                      /场景目录
|-- Scripts                                     /脚本目录
    |-- Controller                              /控制器目录
        |-- Home                                /***Home***模块，项目默认模块 
    |-- Static                                  /存放公共静态类函数目录
    |-- Model                                   /模型。数据库相关操作。
    |-- View                                    /视图目录
        |-- HomeView                            /层次面板与之对应的业务模块
            |-- Config                          /用于存放当前HomeView视图中的所有文本信息变量
```
